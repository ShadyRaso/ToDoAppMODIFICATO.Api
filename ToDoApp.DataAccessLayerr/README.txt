Questo progetto è stato generato via scaffolding tramite il seguente comando: 
Scaffold-DbContext "Server=(localdb)\MSSQLLocalDB;Database=todoapp;Trusted_Connection=True;" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Entities -f

Per eseguirlo in visual studio cliccare su 
tools -> Manager nuget -> Console -> eseguire il comando

Il buon esito dello scaffolding dipende dall'esistenza del db che è stato chiamato "todoapp" e della relativa tabella "todoItems"
